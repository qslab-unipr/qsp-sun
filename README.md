# Quantum State Preparation with $\Lambda$-type operators

## Description
This repository is dedicated to Sun et al.'s QSP algorithm (arXiv 2108.06150) and its optimizations. The folder SUN_v1.0.0 contains the implementation of the original version in the first parametric range defined by Theorem 1. The folder OSUN_v1.0.0 contains the optimized version in the same parametric range.
